# Directory-Manager @ EEP

Basic File System Manager in RAM with basic Linux functions

Currently only directories

## Functions

* help
* exit
* clear
* ls
* ls -l
* mkdir dir_name
* rmdir dir_name
* rm dir_name
* pwd
* cd ..
* cd dir_name
